# Portfolio Website

A personal portfolio website built to showcase my projects, skills, and experience as a Computer Science student. This website highlights my work in software development, web development, and UI/UX design in a clean and responsive layout.

---

## Features

- Fully responsive design for all devices
- Modern and clean UI layout
- Smooth scroll navigation between sections
- Animated sections using AOS library
- Project showcase section
- Contact form for communication
- Organized sections for skills and about me

## Getting Started

To run this project locally:

1. Clone the repository or download the project files  
2. Open the folder in VS Code or any code editor  
3. Open `rishi.html` in your browser  
4. Explore the website  

---

## Usage

- Scroll through the homepage to view all sections  
- Navigate using the top navigation bar  
- View featured projects in the “My Work” section  
- Use the contact form to send a message  
- Click social media icons in the footer to connect  

---

## Technologies Used

- HTML5  
- CSS3  
- Bootstrap 4  
- JavaScript  
- AOS (Animate On Scroll Library)
